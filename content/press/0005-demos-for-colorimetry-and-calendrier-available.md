This year, we have introduced two new libraries to the C# ecosystem so far, which are Colorimetry and Calendrier, which allow you to get information about both a color and a calendar, respectively. For those who are new, a brief explanation about those libraries is mentioned below.

[Colorimetry](https://github.com/Aptivi/Colorimetry) is a library that used code from Terminaux to provide support for more applications, especially when it comes to GUI applications like web apps, native computer apps, and mobile apps, to allow such apps to process colors and convert color models with transformation and color blindness support. [Calendrier](https://github.com/Aptivi/Calendrier) is a library that provides apps an ability to more easily declare a calendar class for multilingual apps.

Today, we have announced new demonstrations for both Colorimetry and Calendrier! The demonstrations can be found in [this link](https://aptivi.github.io/Demos/).

We have added three demonstrations, which are:

* **Calendrier**
  * **Colors - Color Info (manual)**: This demonstrates an ability for Colorimetry to parse a color with a color specifier string that describes a color. You can learn more about the color specifier string [here](https://aptivi.gitbook.io/aptivi/colorimetry-manual/usage/color-sequences/color-model-parsing#supported-color-model-specifiers).
  * **Colors - Color Info (select)**: This demonstrates an ability for Colorimetry to parse a color with your web browser's color wheel function. The color selection dialog box differs in appearance and in behavior, depending on your device.
* **Colorimetry**
  * **Calendar - Calendar Info**: This demonstrates an ability for Calendrier to give you some info about a built-in calendar embedded in the library.

We'll add more demonstrations for our libraries from time to time to showcase more features as demonstrations. These demonstrations will give you a first impression of some of our C# libraries before trying them out in your .NET project.

As updates to our projects continue, we will add more promising and exciting featues, while improving existing ones and fixing bugs to increase both reliability and usability of our projects. Meanwhile, stay tuned for the next release of Terminaux 8.3, which will provide improvements in certain areas, as well as new features, and we promise zero breaking changes when upgrading.
