Aptivi Newsroom was established on April of this year to share news that are related to our projects, such as press releases, new versions, and other statuses, to let users know of the most recent news that happened to them. Meanwhile, the API documentation site for each project that uses C# shows you an overview of all classes and their members, such as properties and methods, along with a short explanation for each item, taken directly from the source code.

However, both of those sites needed some improvements related to look and feel, and Aptivi Blog has recently earned a huge redesign that focused on utilizing responsive design, along with changes to the elements and their look, to make the blog readable across many form factors, such as tablets and phones. You can learn more about the blog redesign [here](/newsroom/press/0024-aptivi-blog-redesign-complete-june-2026).

To extend our efforts to both the Aptivi Newsroom and the API documentation sites, we've decided to expand this rebranding effort to those sites, and we've made this rebranding in appropriate methods, such as different design elements and properties.

We are very excited to announce that the Aptivi Newsroom and API docs have been redesigned! This minor redesign focuses on consistency with the Aptivi Blog that earned its own redesigned style, ensuring that your experience has been enhanced across several form factors.

We have employed the current design of Aptivi Blog, along with appropriate changes made to different elements, based on purpose, such as the readjusted left sidebar that has managed to use the `flex` style instead of the clunky `fixed` position declaration in the CSS design.

We have also done the necessary adjustments to ensure better user experience across all form factors, especially when we take mobile phones into account. Visual representations can be viewed in the Aptivi Blog article for this event.

You can now view the renewed design of both the Aptivi Newsroom and the API documentation for each C# project's sites.
